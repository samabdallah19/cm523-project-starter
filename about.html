For this project I wanted to create a fun way to learn about the geography of the United States. The United States is a massive country with all 50 states having their own culture and defining aspects. Knowing this, I wanted to create a way for people to learn about where these states lie and other facts about them. I believe that having a better grasp on the geography of the United States opens up a new world of opportunity for human interaction and understanding. This is a low stakes game meant to be a learning opportunity. Despite Slick not being initialized in my code, the Slick set up was still incorporated in my code to get my quiz questions to be laid out the way that they are. The Interactive and Responsive SVG Map of the US States and Capitals by Website Beaver really opened up these learning opportunities is an invaluable tool to have the state information at the user's disposal.
